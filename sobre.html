Como surgiu este Projeto?

Este Projeto nasceu de uma ideia, á partir de um siples roteiro de estudos. Desenvolvido para ser o estágio final de um aprendizado nos fundamentos básicos do Front-End; usando apenas HTML&CSS puro.Eu peço o programador ou espectador que contempla este site perdoe as inconcistencias de erros ortográficos.Uma versão com componentes em Java Script será lançada futuramente.

Mas porque o tema deste site é Motivação.A motivação é um ingrediente chave para nossas Vidas e nosso trabalho no dia a dia. É partir dela que conseguimos estabelecer um ideial, um objetivo e assim focar naquilo que sonhamos.Este site é para voce.
